📖 Project Overview

The Battery Management System (BMS) is an advanced IoT-based monitoring solution designed to provide real-time insights into battery performance and health. This system combines embedded systems, IoT connectivity, and web technologies to create a comprehensive battery monitoring platform.

Our BMS continuously monitors critical battery parameters including voltage, current, temperature, and state of charge (SOC). The system provides instant alerts for potentially dangerous conditions and maintains historical data for analysis and predictive maintenance.

🎯 Key Features

  • Real-time monitoring of voltage, current, temperature, and SOC
  • Live data updates every 2 seconds
  • Automated safety alerts and warnings
  • IoT connectivity for remote monitoring
  • Web-based dashboard accessible from any device
  • Historical data logging and analysis
  • Battery health monitoring
  • User-friendly interface with visual indicators

Data Flow

1. Data Collection: Sensors continuously measure battery parameters (voltage, current, temperature).

2. Processing: STM32 microcontroller processes raw sensor data, performs calculations, and ensures data accuracy.

3. Transmission: Processed data is sent to ESP32 via UART communication.

4. IoT Layer: ESP32 transmits data to the cloud/server using MQTT or HTTP protocol over WiFi.

5. Server Processing: Python Flask server receives data, stores it in database, and serves it through REST API.

6. Visualization: Web dashboard fetches data via API and displays it with real-time updates every 2 seconds.
